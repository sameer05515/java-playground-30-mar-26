A **palindrome** is a word, number, phrase, or sequence that reads the same **forward and backward** (ignoring spaces,
punctuation, and capitalization for phrases).

### ✅ Examples:

* Words: `madam`, `level`, `radar`
* Numbers: `121`, `1331`, `12321`
* Phrases: `"A man a plan a canal Panama"` (ignoring spaces and case)

### ❌ Non-palindromes:

* `hello`, `world`, `1234`
